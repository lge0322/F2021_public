Regular Expressions

A brief introduction to regular expressions, aka regex, a common and powerful tool in programming (this also exists in Python). A regular expression is a form of syntax that defines a "pattern." This pattern can then be used to find substrings that match certain criteria. There are many types of patterns, and the syntax can get hard to read. The two buttons below each request a string from the user and then display the results of matching various regex values with those strings. Rather than get into the nitty gritty of explaining the syntax, each regex has an explanation attatched to it saying what that regex looks for.
