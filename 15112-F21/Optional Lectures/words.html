Programming with JS

Playing Word Search

A demonstration of how to do many of the tasks you have learned so far in 112 using JS, in a way that interacts with the user. This interactive game of word search uses the word search algorithm, loops, conditionals, strings, lists, sets, and several forms of getting user input / giving the user feedback.

When the button below is clicked, a function is called that creates a string containing a bunch of HTML tags which represent the cells of the board. The HTML is then injected into the page, along with the instructions and a list of words remaining to be found. The board, its dimensions, and the set of remaining words are global variables. Each of the cells has an onclick attribute which is how the gameplay is formed. The JS also makes use of the window.prompt(), which creates a popup window asking for a string, and the alert() function which creates a popup messagebox.
