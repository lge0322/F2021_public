Objects in JS

Classes, Methods, Attributes and Inheritence

A demonstration of some of the OOP features in JS, many of which mimick those of Python. The JS code for this page defines a Rectangle class in a way that is very similar to how it is done in Python. There is a constructor (i.e. __init__), attributes, and methods. One difference is that there is no "self" parameter. In JS, it is called "this", and it is accessible without making it a parameter. The constructor takes in and stores the top left x-coordinate, top-left y-coordinate, width and height of the rectangle. There is also a toString() method which converts a Rectangle object to a string, an area() method which returns the area of a rectangle, and a contains() method which takes in an (x, y) coordinate and returns True if that coordinate is inside of the Rectangle.

The JS code also defineds a Square class which extends Rectangle. The way this is often phrased is that Square is a 'subclass' of Rectangle, or that it 'inherits' from Rectangle. This means that every method written inside of Rectangle is implicitly copied to Square, but we can also overwrite some of the methods and define new ones. Square has a new constructor which only takes in the x, y, and size instead of x, y width and height. However, this constructor reuses Rectangle's constructor via the super() keyword. It also overwrites toString() with a new definition. The area() and contains() methods are not mentioned, so Square has the same definitions for these methods as Rectangle. There is also a new method for Squares only: scale() takes in a number and resizes the square by that number.

Yes, HTML has a canvas and JS can draw on it

Below is a canvas object, created via the <canvas></canvas> tag. It has a certain id, so we can use JS to look it up. This canvas object has several methods that we can use to draw on it, much like with cmu_112_graphics. The first 2 buttons below create new Rectangle or Square objects based on user input, and append it to a list stored in a global variable. When either type is created, the constructor calls the randomColor() function defined in the JS code to get a new color and assigns it to that Rectangle/Square. Every time a change is made, a funciton is called that loops over the list and uses the canvas to draw them based on their x, y, width, height and color. There is a 3rd button which takes in a point and returns all of the Rectangles or Squares that contain that point (displayed via the toString method). There is also a 4th button which takes in a number and destructively scales all the squares in the list by that factor.
